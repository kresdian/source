Sertakan CSS & JavaScript
Sertakan file Material Lite CSS dan JavaScript di setiap halaman HTML di proyek Anda. Kami menyarankan Anda menggunakan file yang dihosting di CDN kami 
<link rel="stylesheet" href="./material.min.css">
<script src="./material.min.js"></script>
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Aturan Umum
    Secara umum, ikuti langkah-langkah dasar ini untuk menggunakan komponen MDL di halaman HTML Anda:
  1. Mulailah dengan elemen HTML standar, seperti <button>,<div>, atau <ul>, tergantung pada komponen MDL yang ingin Anda gunakan. Ini menetapkan elemen di halaman dan menyiapkannya untuk modifikasi MDL.
  2. Tambahkan satu atau lebih kelas CSS spesifik MDL ke elemen, seperti mdl-buttonatau mdl-tabs__panel, lagi tergantung pada komponen. Kelas menerapkan peningkatan MDL ke elemen dan mengubahnya menjadi komponen MDL.
    3. Ingatlah untuk menyertakan tag meta viewport dalam dokumen Anda sehingga perangkat seluler akan merender dengan benar. 
<!doctype html>
<html lang="en">
  <head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <!-- Bootstrap CSS -->
    <link rel="stylesheet" href="material.min.css">

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">
    <title>Hello, world!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    
  
    <!-- Optional JavaScript -->
    <!-- jQuery first, then Popper.js, then Bootstrap JS -->
    <script src="material.min.js"></script> </body>
</html>
